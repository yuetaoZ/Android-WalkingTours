# Android-WalkingTours
This Android app is used to give real-time tour attractions information of the Chicago area, notifications will pop-up based on Google Maps location.
<p align="center">
  <img src="https://github.com/yuetaoZ/Android-WalkingTours/blob/master/app/src/main/res/raw/demo_main2.png" width="300" title="main screen2">
</p>
A notification will pop-up when the user is walking near to an attraction. By clicking on the notification, a website about the attraction will be displayed.
<p align="center">
  <img src="https://github.com/yuetaoZ/Android-WalkingTours/blob/master/app/src/main/res/raw/demo_notification.png" width="350" alt="notification">
  <img src="https://github.com/yuetaoZ/Android-WalkingTours/blob/master/app/src/main/res/raw/demo_notification_detail.png" width="350" title="attraction details">
</p>
